# SujaySarma.Data.Azure.Tables
The .Net 5.0+ version of the previous Azure Tables SDK
---

## NuGet Package link:
https://www.nuget.org/packages/SujaySarma.Data.Azure.Tables/

## Pre-requisites:

- .NET 6.0

Works with: 
- Azure Storage's Tables service
- Azure Cosmos DB with Tables API
- Azurite Storage Emulator's Tables service
